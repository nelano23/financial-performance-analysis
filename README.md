# financial-performance-analysis
Overview
Analysis of 10,800+ Superstore transactions to identify regional and category-level sales and profit trends using Python and pandas.

Tools Used
Python
pandas
matplotlib
Jupyter Notebook
Key Findings
The West region leads in both sales and profit, while Central has surprisingly low profit despite being the second highest in sales.
Furniture generates high sales but almost no profit compared to Technology and Office Supplies.
Technology is the most profitable category, making nearly 8x more profit than Furniture despite similar sales volume.
